# Genshin_book

## Stack:
- Network connection - Supabase(Ktor), Okhttp, Retrofit
- Database - Realm, Room
- Asynchronous - RxJava 2, Coroutines, Flow
- UI - Compose,XML
- UI navigation - Voyager,Cicerone
